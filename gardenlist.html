Garden App

(2017) see live demo

Mean stack application. Built with Node.js, Angularjs, MongoDB, Passportjs, CSS3, Gulp, Postman, Bcrypt.

Currated by Elena; this simple app allows a user to create an account, and manage their collection of plants they work with. Two separate Mongo DB collections are implemented to store bcrypted User Accounts and plant items generated by each user.

Passport.js is used to authenticte user, create a session cookie, attach req.user when a user is logged into a session, which will be implemented to allow Angular.js to conditionally hide/show features.

Designed with an Angular.js factory service; allows the List controller to make AJAX CRUD requests to the app's API endpoints, $routeProvider to serve partial html views and ngShow as provides a method of condition to show or hide UI features ngRepeat is used to construct the plant items to the list display.
